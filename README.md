# 🥖 Catálogo de Preventa - Panadería Familiar

Aplicación web móvil-first diseñada específicamente para optimizar la logística de preventa y distribución diaria de productos de panificación artesanal (pan, facturas, bizcochitos). 

El sistema permite a los clientes armar su pedido con un día de anticipación y consolidar la orden de compra empaquetada directamente hacia el canal de WhatsApp del repartidor, eliminando la necesidad de bases de datos complejas o pasarelas de pago costosas en la fase de validación comercial.

---

## ⚡ Características Principales

*   **Rendimiento Ultra-Light:** Desarrollado con Vanilla JavaScript y CSS puro bajo el empaquetador **Vite**. Minimiza el consumo de datos móviles para clientes en la vía pública.
*   **Mobile-First Design:** Interfaz optimizada exclusivamente para pantallas táctiles y navegación fluida desde celulares.
*   **Reactividad Local:** Cálculo de totales y control de stock/cantidades en tiempo real en el cliente.
*   **Integración de API de WhatsApp:** Formateo automático de strings codificados para generar hojas de ruta limpias y estandarizadas al recibir los mensajes.

---

## 🛠️ Stack Tecnológico

*   **Build Tool:** Vite
*   **Lenguaje:** JavaScript (ES6+)
*   **Estilos:** CSS3 nativo utilizando Variables CSS
*   **Arquitectura:** SPA (Single Page Application) estática
